# email-generator
A tool to generate html email.

## Usage

### install

```bash
yarn install

gem install compass
```

### watch

```bash
yarn watch
```

### build

```bash
yarn build
```

### dirs

```js
const dirs = {
  src: './src',
  i18n: './src/i18n',
  partials: './src/partials',
  styles: './src/styles',
  templates: './src/templates',
  dest: './dist',
  build: './build'
}
```
